Company

1.       Introduction

Company System is part of LAS Database consisting of details of all companies we deal with. This will include details of our own companies, clients, vendors, service providers and others companies of interest.

LAS database is the main information system of LAS Engineers & Consultants Pvt Ltd. It contains information about Employees, Projects, Companies. Besides, it also contains the document (drawings, Specifications, etc.) details of all on-going projects and Manhour utilization of all employees on projects.

6.       Database Roles

Following are the Database Roles defined for LAS Database System.

         Database Manager

         Project Manager

         HR Manager

         Lead Engineer

         Employee

Database Manager will have full access to the database system. He will be able to create new database tables, schemas. He will be able to view, add, modify and delete all tables and contents.

Project Manager will have access to add, modify and delete the data in the tables depending upon the roles specified for Project Manager on each of the table. He will also have all the privileges of Lead Engineer.

Lead Engineer will have access to add, modify and delete the data in the tables depending upon the roles specified for Lead Engineer on each of the table.

HR Manager will have access to add, modify and delete the data in the tables depending upon the roles specified for HR Manager on each of the table.

Employee will have access to add, modify and delete the data in the tables depending upon the roles specified for Employee on each of the table.

It is to be noted that the Roles specified above are only for the database access and it has no relation with the designation of the employee.
